Airbnb-Power-BI-Dashboard

Project Overview
This project focuses on analyzing Airbnb listing data using Power BI to uncover insights related to pricing, room types, and location trends. The goal is to transform raw data into an interactive and visually appealing dashboard that supports better understanding and decision-making.

Objectives
To analyze the distribution of Airbnb listings across different cities
To study pricing patterns based on location and room type
To understand customer preferences through ratings
To present insights using an interactive dashboard
Tools Used
Power BI
Microsoft Excel
Dataset Details
Dataset: Airbnb Listings
Rows: 1000+
Columns: Includes city, neighbourhood, price, room type, ratings, etc.
Data Cleaning Steps
Removed unnecessary columns
Cleaned price column (removed symbols and converted to numeric)
Handled missing values
Changed data types for accurate analysis
Removed duplicate records
Dashboard Features
KPI Cards: Total Listings, Average Price, Average Rating
Bar Chart: Average Price by City
Pie Chart: Distribution of Room Types
Slicer: City filter for interactive analysis
Insights
Certain cities have significantly higher average prices
Entire homes/apartments are more commonly listed compared to other room types
Pricing varies based on location and availability
Ratings help identify more preferred listings

Conclusion
The dashboard provides a clear and interactive view of Airbnb data, helping to identify pricing trends, popular room types, and location-based differences. This project demonstrates the use of Power BI for effective data visualization and analysis.

Note
This project is created for learning purposes to practice data cleaning, visualization, and dashboard creation using Power BI.
